# AI Use Statement
In this project, AI assistants (Gemini and ChatGPT) were used as support tools during the development, testing, and reporting stages.
AI support was used for:
* Understanding and implementing reinforcement learning methods,
* Fixing Python programming errors and dependency issues,
* Understanding simulator outputs and error messages,
* Improving code readability and organization,
* Assisting with report writing and translation tasks.
However, all final implementation decisions, algorithm development processes, experiment designs, parameter selections, result analyses, and report contents were reviewed and verified by the project team.
